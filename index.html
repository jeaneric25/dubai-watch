What we track & how we do it

Dubai Watch monitors 10 indicators to score whether conditions are safe to return to Dubai during the Iran-UAE conflict (active since Feb 28, 2026). Each indicator is scored 0-100, weighted, and combined into a single composite score.

Scoring zones

TAX TETRIS (0-32): Stay in Europe, manage your tax days carefully.
RECON MODE (33-65): Consider a short recon trip, keep European options open.
FIRST CLASS HOME (66-100): Conditions safe enough to return.

Composite formula

Composite = Sum(score_i x weight_i) / Sum(weight_i) — Total weight: 77

1. Iranian launches (weight: 10)

Source: @modgovae on X (official UAE Ministry of Defence). We count ballistic missiles + UAVs reported in daily engagement tweets.
Scoring: Binary — any launches > 0 = score 0, zero = score 100.

2. UAE shelter alerts (weight: 10)

Source: @NCEMAUAE on X (National Emergency Crisis and Disaster Management Authority). We count tweets containing shelter/alert/warning keywords.
Scoring: Binary — any alerts > 0 = score 0, zero = score 100.

3. DXB flights (weight: 9)

Source: FlightRadar24 API (flight-summary/light endpoint). Counts departures + arrivals combined.
Baseline: 1,200 movements/day (pre-conflict DXB average).
Scoring: score = min(100, current / baseline x 100)

4. Brent crude (weight: 8)

Source: Yahoo Finance / CNBC.
Baseline: $77.50/bbl (pre-conflict average).
Scoring: Penalises deviation from baseline. score = max(0, 100 - |deviation| x 200)

5. VIX (weight: 5)

Source: Yahoo Finance.
Baseline: 16.5 (pre-conflict average).
Scoring: If VIX ≤ baseline: 100. Otherwise: score = max(0, 100 - (VIX - 16.5) / 16.5 x 200)

6. UAE govt posture (weight: 8)

Source: @modgovae on X. Total daily tweet count = communiqués per day. Higher count = more active conflict communications.
Scoring: score = max(0, 100 - count x 10)

7. Saudi govt posture (weight: 8)

Source: @modgovksa on X (Saudi Ministry of Defence). Same methodology as UAE posture.
Scoring: score = max(0, 100 - count x 10)

8. French advisory (weight: 6)

Source: diplomatie.gouv.fr (French Ministry of Foreign Affairs).
Scoring: Normal = 100, Vigilance renforcée = 60, Déconseillé = 20, Formellement déconseillé = 0.

9. Embassy status (weight: 7)

Source: Manual check of G7 + P5 embassy websites.
Scoring: All open = 100, Some closed = 40, All closed = 0.

10. Civil normalcy (weight: 6)

Source: KHDA (Dubai schools authority) and government announcements.
Scoring: Both open = 100, Schools or offices closed = 40, Both closed = 0.
